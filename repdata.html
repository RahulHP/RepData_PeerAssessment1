Devise a strategy for filling in all of the missing values in the dataset.

We will replace the missing values in an interval by the mean of that 5-minute interval. We first make a copy of the original data which will be used for further calculations.

The loop goes through all the 'step' values of the data. If a value is NA it is replaced by the average of that specific interval (which is found from the 'dailyactivity' variable).

copy=data
for (i in 1:nrow(copy)){
  if (is.na(copy[i,1])){
    row= (copy[i,3]+5)/5
    copy[i,1]=dailyactivity[row,2]
  }
}
